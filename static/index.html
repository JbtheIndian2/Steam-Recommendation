How the match is computed

01 / VECTORIZE

TfidfVectorizer

Every game's genres, Steam-community tags, and multiplayer-relevant categories become tokens. scikit-learn's TfidfVectorizer builds an 8,640 × 310 sparse matrix from them.

02 / WEIGHT

Inverse document frequency

Common tokens like Indie carry little signal since most games have them. Rare tokens like Metroidvania weight higher — TF-IDF handles this automatically per-token.

03 / RANK

NearestNeighbors (cosine)

A fitted sklearn.neighbors.NearestNeighbors index retrieves each game's closest vectors by cosine distance. The badge is that similarity score; "shared" names the top overlapping tokens.
